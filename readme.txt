Description:

La Landing Page de Inteligencia Artificial es una web sencilla diseñada con HTML y CSS  que se presentar como un ejemplo de Landing page o pagina de aterrizaje para la escuela de formacion Peñascal F5 donde se da a conocer un nuevo curso. 

Para poder finalizar el embudo de conversión la web de divide en  tres partes:
Un video introductorio, un texto y el formulario.

Web Map

La pagina web esta formada por dos secciones la Home, Blog y Noticias
Los elementos utilizados en la web han sido:

- Codigo HTML, CSS y Formularios.

HTML:
Etiquetas semanticas,diferentes encabezados,hipervinculos externos e internos de la web.
Incluye enlace a vides y a imagenes.

CSS:

Utilizamos la regla de css de background (color, image)combinando el color blanco del background, con el header de color negro y el footer de color gris y con el boton de enviar formulario de color naranaja, para llamar la atención al usuario.

La web es responsive para tablet, escritorio y movil.

FORMULARIO:

Incluye un formulario  que recoge los datos de usuario (NOmbre,Email, mensaje y el boton de enviar) Con la casilla de verificación de politica de privacidad




NAVBAR

El menu esta formado por un menu de Blog o Home cuando ingreses al blog,  que te invita a regresar a la pagina de origen.


RESPONSIVE
Todas las secciones son responsive.

---------------------------------------JAVESCRIPT-----------------------------------------

Para el apartado de funcionalidades con JavaScript he realizado 5 funcionalidades.

1 - El botón de enviar del formulario se habilitará solo cuando marque el checkbox

2 - Cambio de color de imagen del header segun el color que quieras pasar de negro(original) a azul.(Interacción con los elementos del DOM)

3 - Validación de formulario

4 - Mostrar mensaje de Exito al enviar el mensaje

Todas esta funcionalidades estan en el archivo script.js que se encuentra en la Home. Ya que la todas las funcionalidades se visualizan en la home.html.( ES MUY IMPORTANTE IR COMENTANDO Y DESCOMENTANDO CADA FUNCIONALIDAD PORQUE A VECES NO SE PUEDEN VER.) 

Por otro lado, en el archivo script.js tendra comentado y una breve explicacion de cada funcionalidad.

 LOGIN DE USUARIO:
